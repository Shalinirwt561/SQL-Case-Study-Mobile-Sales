# 📊 SQL Case Study: Mobile Sales Database

## 📌 Project Overview
This project demonstrates SQL skills by creating a mobile sales database including:
- Database creation
- Table relationships (Star Schema)
- Data insertion
- Transaction analysis

## 🧱 Database Structure
- DIM_MANUFACTURER
- DIM_MODEL
- DIM_CUSTOMER
- DIM_LOCATION
- DIM_DATE
- FACT_TRANSACTIONS

## 🔍 Key Concepts Used
- Primary Keys & Foreign Keys
- Joins
- Data Modeling
- Aggregations
- SQL Queries

## 📂 File Included
- MOBILE.sql → Complete database script

## 🎯 Purpose
This project is built to practice SQL and demonstrate data analysis skills for interviews.
